# Space Invaders
Game bắn người ngoài hành tinh cho lớp Lập trình nâng cao

Game sử dụng C++ và thư viện SFML

## How to play
- W, A, S, D to move, Left-click to shoot
- You lose HP colliding with the enemy
- 1, 2, 3 to change the enemies
